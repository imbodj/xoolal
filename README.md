> Premier navigateur web sénégalais sécurisé — © 2024 Ismaïla Mbodji

## 🚀 Présentation

**Xoolal** signifie **"Regarde !"** en wolof 🇸🇳

Navigateur web sécurisé créé au Sénégal avec Python et PyQt5.

## ✅ Fonctionnalités

- 🚫 Bloqueur de publicités (30+ réseaux)
- 🛡️ Isolation des onglets
- 🍪 Gestion des cookies
- 🔍 6 moteurs de recherche
- 🕵️ Navigation privée
- 👶 Mode enfant
- 🌙 Mode sombre
- 🔒 Indicateur HTTPS

---

## 🚀 Installation depuis le code source
```bash
pip install PyQt5 PyQtWebEngine
python3 navigateur.py
```
