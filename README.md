## 🚀 Présentation

**Xoolal** est un navigateur web sécurisé créé au **Sénégal 🇸🇳**,
développé avec Python et PyQt5.

Son nom vient du wolof : **"Xoolal"** signifie **"Regarde !"**

---

## ✅ Fonctionnalités

| Fonctionnalité | Description |
|---|---|
| 🚫 Bloqueur de pubs | Bloque 30+ réseaux publicitaires |
| 🛡️ Isolation des onglets | Chaque onglet est totalement isolé |
| 🍪 Gestion des cookies | Contrôle total sur les cookies |
| 🔍 Moteur au choix | Google, Bing, DuckDuckGo, Qwant... |
| 🕵️ Mode privé | Pas d'historique, pas de cookies |
| 👶 Mode enfant | Sites éducatifs uniquement |
| 🌙 Mode sombre | Interface sombre |
| 🔒 HTTPS | Indicateur de sécurité |
| 🇸🇳 Thème sénégalais | Couleurs vert-jaune-rouge |

## 📦 Installation

### Linux (Ubuntu/Debian)
```bash
sudo dpkg -i xoolal_1.4.0_all.deb
```

### Depuis le code source
```bash
# Installer les dépendances
pip install PyQt5 PyQtWebEngine

# Lancer Xoolal
cd mon-navigateur
python3 navigateur.py
```
